# RobuBubble

Earn Free Robux akışına odaklı premium görev paneli.

## Komutlar (Net)
- **Start Command:** `npm start`
- **Dev Command:** `npm run dev`
- **Build Command:** `npm run build`

## Kurulum
```bash
npm install
npm start
```

Sunucu: `http://localhost:3000`

## Build çıktısı
```bash
npm run build
```
Bu komut `index.html`, `robots.txt`, `sitemap.xml` dosyalarını `dist/` klasörüne kopyalar.

## API Endpointleri
- `GET /api/health` → servis sağlık kontrolü
- `GET /api/tasks` → örnek site ziyaret görevleri

## Google'da Siteyi Çıkarmak (SEO)
1. Domain bağla (ör. `https://robububble.com`).
2. Google Search Console'a siteyi ekle: https://search.google.com/search-console
3. Doğrulama yap (DNS / HTML file / meta tag).
4. `https://robububble.com/sitemap.xml` adresini Search Console'da gönder.
5. URL Denetimi ile önemli sayfalar için dizine ekleme isteği gönder.
