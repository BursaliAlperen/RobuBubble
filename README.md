# RobuBubble

ZerAds offerwall odaklı (tek kazanç akışı) premium Robux görev paneli.

## Frontend + Backend Kurulum

### 1) Bağımlılıkları kur
```bash
npm install
```

### 2) Development / Start
```bash
npm run dev
```
veya
```bash
npm start
```

Sunucu: `http://localhost:3000`

### 3) Build Command
```bash
npm run build
```
Bu komut `index.html`, `robots.txt`, `sitemap.xml` dosyalarını `dist/` klasörüne kopyalar.

## API Endpointleri
- `GET /api/health` → servis sağlık kontrolü
- `GET /api/tasks` → örnek ZerAds görevleri

## Google'da Siteyi Çıkarmak (SEO)
1. Domain bağla (ör. `https://robububble.com`).
2. Google Search Console'a siteyi ekle: https://search.google.com/search-console
3. Doğrulama yap (DNS / HTML file / meta tag).
4. `https://robububble.com/sitemap.xml` adresini Search Console'da gönder.
5. Önemli sayfalar için “URL Denetimi > Dizine eklenmesini iste”.
6. Sayfa title/description/keywords ve schema markup güncel tut.
